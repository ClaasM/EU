Über diesen Umrechner

Dieser Speicher und Datenumrechner beherrscht alle Gängigen Größen und Einheiten von Daten. Dazu gehören 1000^n-fache von Bits wie das Kilobit (Kb oder Kbit) oder das Megabit (Mb oder Mbit), 1000^n-fache von Bytes wie das Kilobyte (KB) oder das Megabayte (MB), 1024^n-fache von Bits wie das Kibibit (Kib oder Kibit) oder das Mebibit (Mib oder Mibit) sowie 1024^n-fache von Bytes wie das Kibibyte (KiB) oder das Mebibyte (MiB)

Speicher und Daten

Ein Datenspeicher oder Speichermedium dient zur Speicherung von Daten. Der Begriff Speichermedium wird auch als Synonym für einen konkreten Datenträger verwendet. Konkret dient ein Datenträger oder Speichermedium dem Speichern von Daten jeglicher Art (Musik, Filme etc.). Bei elektronisch-digitaler Speicherung unterscheidet man zwischen flüchtigen Speichern (z.b. RAM), permanenten Speichern (z.b. ROM) und semi-permanenten Speichern, wie zum Beispiel denen in USB-Sticks oder auf Festplatten. Speichermedien für den Konsumermarkt liegen meist im Bereich Gigabyte bis Terabyte.

Bit

Das Bit wird in der Informationstechnik, Informatik und Nachrichtentechnik als Maßeinheit für den Informationsgehalt und digital repräsentierte Datenmengen verwendet. Außerdem wird es als Bezeichnung einer Stelle einer Binärzahl oder einer anderen Gruppe binärer Stellen verwendet. Zu den vom Bit abgeleiteten Größen in diesem Umrechner zählen:

Einheit Bit

Kilobit (Kb, Kbit) 1e3
Megabit (Mb, Mbit) 1e6
Gigabit (Gb, Gbit) 1e9
Terabit (Tb, Tbit) 1e12
Petabit (Pb, Pbit) 1e15
Exabit (Eb, Ebit) 1e18
Zettabit (Zb, Zbit) 1e21
Yottabit (Yb, Ybit) 1e24

Neben den vielfachen von 1000-fachen des Bits gibt es noch die IEC-Präfixe. Diese stellen das 1024^n-fache eines Bits dar und werden oft fälschlicherweise mit den obigen Einheiten verwechselt. Die Einheiten der IEC Präfixe, die mit diesem Umrechner umgerechnet werden können, sind das Kibibit (Kibit, kib), das Mebibit (Mib, Mibit), das Gibibit (Gib, Gibit), das Tebibit (Tib, Tibit), das Pebibit (Pib, Pibit), Exbibit (Eib, Eibit), das Zebibit (Zib, Zibit) und das Yobibit (Yib, Yibit). Byte

Das Byte ist eine Maßeinheit der Informatik, das für eine Folge von 8 Bit steht. Es entstammt der früher benötigten Anzahl Bits zur Kodierung eines Schriftzeichens und wird manchmal auch Oktett genannt. Es ist eine Zweierpotenz (2^3) und ist das kleineste adressierbare Element in vielen Rechnerarchitekturen. Ein Nibble ist ein halbes Byte, also 4 Bit. Zu den mit diesem Umrechner umrechenbaren, vom Byte abgeleiteten Einheiten Zählen das Nibble, das Kilobyte (KB), das Megabyte (MB), das Gigabyte (GB), das Terabyte (TB), das Petabyte (PB), Exabyte (EB), das Zettabyte (ZB) und das Yottabyte (YB). Diese entsprechen in Byte:
